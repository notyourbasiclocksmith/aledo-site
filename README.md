# Aledo Locksmith — Professional Locksmith in Aledo, TX

Bilingual (EN/ES) locksmith website for Aledo, TX built with Next.js 14, Tailwind CSS, and TypeScript.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — root redirects to `/en`.

## Build & Export

```bash
npm run build
```

Static output is written to the `out/` directory.

## Tech Stack

- **Framework:** Next.js 14 (App Router, Static Export)
- **Styling:** Tailwind CSS 3.4
- **Language:** TypeScript
- **Icons:** Lucide React
- **Form Backend:** FormsAI
- **Bot Protection:** reCAPTCHA v3
